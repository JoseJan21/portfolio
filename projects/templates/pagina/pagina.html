Descripción de la web

Este es un sitio web con características similares a un blog. Utiliza Python y el framework Django para su desarrollo y ofrece una interfaz de administrador de Django que permite crear y gestionar diferentes vistas en la aplicación, articulos, categorias y usuarios

Además, la aplicación cuenta con una sección de creación de artículos, lo que permite a los usuarios publicar contenido en el sitio web

También tiene una sección de registro y login, lo que permite a los usuarios registrarse y acceder a su cuenta para crear sus propios artículos. Y para aquellos que no quieran registrarse hay usuarios de prueba, listos para probar la web.

En resumen, esta aplicación es un ejemplo de una aplicación web dinámica y personalizable, que combina las características de un blog con la funcionalidad de una interfaz de administrador y seguridad de usuario.
